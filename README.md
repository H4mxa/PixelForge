> **A fast, modular image processing engine in C.**  
> Apply filters to 256Ã—256 BMPs: B&W, invert, flip, merge, shrink, edge detection.

---

## Features

- `b` Black & White
- `i` Invert
- `d/l` Darken / Lighten
- `h/v` Flip Horizontal / Vertical
- `m` Merge two images
- `s` Shrink (1/2, 1/3, 1/4)
- `t` Edge detection

---

## Build & Run

```bash
make
./build/pixelforge
```
